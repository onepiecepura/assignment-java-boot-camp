## init project
1. Clone repository
    ```
    git clone git@github.com:onepiecepura/assignment-java-boot-camp.git
    ```

2. Checkout to develop branch
    ```
    git checkout develop
    ```

3. Build project command
    ``` shell
    ./mvnw clean install
   ./mvnw clean test
   ./mvnw clean package
   ./mvnw spring-boot:run
    ```
